Instrukcja Obsługi

O mate-chess

Funkcjonalności

  • Gra z AI na poziomach: łatwy, średni, trudny.
  • Rozgrywki online z czatem i globalnymi rankingami.
  • Analiza partii: przegląd ruchów, wykrywanie błędów, sugestie.
  • Obsługa na web, desktopie i urządzeniach mobilnych.
  • Panel administracyjny z logami i zarządzaniem graczami.

Technologie

  • Node.js – serwer i zarządzanie pakietami.
  • Rust – wydajny silnik szachowy.
  • React – nowoczesny interfejs użytkownika.
  • TypeScript – typowany kod.
  • Express – routing i sesje gry.
  • pnpm – szybki menedżer pakietów.
  • Git – kontrola wersji i współpraca.

Instalacja

Wymagania

  • Node.js (wersja 16 lub wyższa)
  • Rust (stable channel)
  • Git (do klonowania repozytorium)
  • pnpm (menedżer pakietów)
  • ts-node (do uruchamiania TypeScript)

Kroki Instalacji

  1. git clone https://github.com/matiqueue/mate-chess.pl.git
  2. cd mate-chess.pl
  3. pnpm install
  4. Środowisko:
    • macOS: pnpm env:mac
    • Windows: pnpm env:win
    • Linux: pnpm env:linux
  5. Tryb deweloperski: 
    pnpm dev
  6. Budowanie: 
    pnpm build
  7. Start (np. web): 
    cd apps/web && pnpm start

Docker: 

docker run -p 3000:3000 matiqueue/mate-chess:latest
